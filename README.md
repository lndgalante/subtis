## Development


1) Install bun runtime

```bash
> curl -fsSL https://bun.sh/install | bash
```

or update it

```bash
> bun update
```

2. Install project dependencies

```bash
> bun install
```

3) Run any of the desired packages

```bash
> bun dev:api // needed for web and cli
> bun dev:web
> bun dev:cli
> bun dev:indexer
```

4. Test your code in watch mode

```bash
> bun test:watch
```
